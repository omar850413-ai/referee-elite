# Guía de Carga a GitHub: Referee Elite

Sigue estos pasos para subir tu código a GitHub y terminar la configuración de App Hosting.

## 1. Preparación del Código
1. Haz clic en **"Zip and Download"** en Firebase Studio.
2. Extrae el contenido del ZIP en una carpeta nueva en tu PC.

## 2. En GitHub.com
1. Crea un nuevo repositorio llamado `referee-elite`.
2. No marques ninguna casilla de inicialización (README, etc.).
3. Haz clic en el enlace que dice **"uploading an existing file"** en la pantalla principal del repositorio vacío.
4. Selecciona todos los archivos de tu carpeta local (incluyendo carpetas como `src`, `public`, etc.) y arrástralos a la ventana del navegador.
5. Haz clic en **"Commit changes"**.

## 3. En la Consola de Firebase
1. Regresa al flujo de configuración de **App Hosting**.
2. Selecciona tu repositorio recién creado.
3. Configura el ID del backend (ej: `referee-elite-web`).
4. Haz clic en **"Finalizar e implementar"**.

Una vez que termine la implementación, Firebase te dará la URL pública de tu aplicación.
