# **App Name**: Referee's Edge

## Core Features:

- Score Tracking: Record and display scores for home and away teams.
- Timer Control: Start, pause, and reset the game timer with accurate timestamps.
- Incident Logging: Log all game incidents, such as goals, fouls, and cards, with timestamps.
- Real-Time Annotations: Allow the judge to write timestamped annotations.
- PDF Report Generation: Generate a detailed PDF report of the game, including scores, incidents, and annotations.
- Data Synchronization: Synchronize match data across devices using Firestore for real-time updates and backup.

## Style Guidelines:

- Primary color: Cyan (#06B6D4) to give a modern, clean aesthetic.
- Background color: Light cyan (#E0F7FA) for a soft, unobtrusive backdrop.
- Accent color: Orange (#F97316) to highlight important actions and annotations.
- Body and headline font: 'Inter', a sans-serif font for clear readability.
- Use consistent and clear icons for game actions, with color-coding for different incident types.
- Use a responsive layout that adapts to different screen sizes, with clearly defined sections for different functionalities.
- Subtle transitions and animations when updating scores or logging incidents.
