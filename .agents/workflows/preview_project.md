# Preview Project

Follow these steps to preview the application locally.

## 1. Start the Dev Server
Run the Next.js dev server from the project root. On Windows (due to PowerShell Execution Policy restrictions), run the `.cmd` version:
```bash
npm.cmd run dev
```
*(On macOS/Linux: `npm run dev`)*

## 2. Verify Port & URL
Wait for the output to show that the server is ready:
- The server usually runs on: [http://localhost:3000](http://localhost:3000)

## 3. Test the Application
Open the preview URL in your browser to verify that the main UI loads correctly and that pages compile.
